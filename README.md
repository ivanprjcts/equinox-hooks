# equinox-hooks

## Prerrequisites ##

* Python 2.7.10+
* Create virtualenv. http://docs.python-guide.org/en/latest/dev/virtualenvs/
* Activate virtualenv.
* Install dependencias.
```
pip install -r requirements.txt
```


## Create DB ##
```
python manage.py migrate hooks
```
## Run server ##
```
python manage.py runserver localhost:8080
```
